Install recommended wordpress.org plugins easy way.

== Description ==

One plugin to rule them all, Uli Plugin Manager is a plugin that present a list of recommended plugins of wordpress.org, 
to make your life easier for install.


== Installation ==

1. Upload `uli-plugin-manager.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Navigate to `Plugins -> Install Plugins` to see the full list of recommended plugins.

== Frequently Asked Questions ==
= Who can see Uli Plugin Manager's menu? =
Any Wordpress user with `manage_options` capabillities.
